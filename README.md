# CaliburnMicro-Calculator
Простой калькулятор с использованием библиотеки Caliburn.Micro (WPF with MVVM)

Caliburn.Micro версия: 3.2 (Устанавливается с Nuget в Visual Studio)

Функционал: +, -, *, /
